Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
metamaven:help Display help information on meta-maven-plugin-maven-plugin.
Call mvn metamaven:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
metamaven:meta-meta-plugin Generates the code for implementing a meta-plugin mojo, using Maven native Plugin configuration in from POM. When executed as a plugin, the generated mojo executes multiple Maven plugins in order.

If the configured plugins bind to multiple Maven phases, the meta-plugin will generate a separate mojo for each phase.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.9
JDK 11

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>meta-maven-plugin</groupId>
          <artifactId>meta-maven-plugin-maven-plugin</artifactId>
          <version>0.1.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>meta-maven-plugin</groupId>
        <artifactId>meta-maven-plugin-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
